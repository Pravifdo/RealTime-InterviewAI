Error: "Mixed Content: blocked HTTP request from HTTPS page"
Fix: Make sure frontend .env.local has HTTPS backend URL (ngrok)

Error: "Failed to fetch templates"
Fix: 
  1. Check backend ngrok tunnel is running
  2. Verify .env.local has correct backend URL
  3. Restart frontend

Error: "WebSocket connection failed"
Fix:
  1. Backend is configured for all origins (*)
  2. Check both ngrok tunnels are active
  3. Clear browser cache (Ctrl+Shift+R)

Error: "ERR_NGROK_6024" (session limit)
Fix: Free ngrok = 1 tunnel at a time
  1. Get auth token: ngrok authtoken YOUR_TOKEN
  2. Or use paid plan for multiple tunnels

Error: Camera/Mic not working
Fix:
  1. Click lock icon in browser address bar
  2. Allow camera and microphone
  3. Refresh page

Error: Participant can't join room
Fix:
  1. Make sure Room ID is correct
  2. Check interviewer created room first
  3. Verify both using same frontend ngrok URL

   - Be ready to restart and update URLs

   - Test on same WiFi network using local IP

   - Use Chrome or Edge (best WebRTC support)
   - Avoid Firefox (may have issues)

6. Gemini AI Key:
   - If not added, system uses keyword matching
   - Add to backend/node/.env: GEMINI_API_KEY=xxx
